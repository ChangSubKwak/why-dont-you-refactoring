# why-dont-you-refactoring

### SimpleCalculatorWithBadSmell 의 기능
- 두 개의 피연산자(정수)와 한 개의 연산자(사칙연산)에 대해서 계산 가능한 프로그램
    ```text
    1 + 2 // Keyboard Input
    3     // Print
    ```
    ```text
    1 - 2 // Keyboard Input
    -1     // Print
    ```
    ```text
    1 * 2 // Keyboard Input
    2     // Print
    ```
    ```text
    6 / 2 // Keyboard Input
    3     // Print
    ```
